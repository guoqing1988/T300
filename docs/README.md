## Remote

how to control with 10 keys

- 'F' forward
- 'B' backward
- 'L' turn left
- 'R' turn right
- 'M' select (menu)
- 'S' start
- 'T' triangle
- 'C' circle
- 'X' button X
- 'Q' sQuare
